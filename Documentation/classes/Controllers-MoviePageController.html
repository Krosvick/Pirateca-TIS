MoviePageController extends BaseController
in package
  • Application

MoviePageController.php : 17

Abstract class representing a base controller.

This class contains various methods for rendering views, redirecting users, returning JSON data, and managing middleware.

Example Usage:

$controller = new BaseController($container, $routeParams);
$controller->render('home', ['title' => 'Welcome']);
$controller->redirect('/login');
$controller->json(['message' => 'Success']);
$controller->registerMiddleware(new AuthMiddleware());
$middleware = $controller->getMiddleware();

moviePage()

MoviePageController.php : 47

Retrieves the movie data from the database using the moviesDAO class and the provided movie ID.

public moviePage(int $id[, mixed $offset = 0 ]) : void

It also retrieves the ratings data using the ratingsDAO class. If the movie data is not found, it aborts the response with a 404 status code. Finally, it renders the movie page template with the retrieved movie and ratings data.
