Router.php : 18

Router class for handling routes and dispatching requests to controllers and actions.

The Router class is responsible for adding routes for different HTTP methods (GET, POST, PUT, DELETE), matching a given URL and HTTP method to a route, extracting values for dynamic segments in the URL, and dispatching the request to the appropriate controller and action.
